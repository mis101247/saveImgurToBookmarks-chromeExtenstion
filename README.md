## About The Project

- 將喜歡的Imgur圖片儲存到你的書籤列中
- Save Imgur image URL to your bookmarks.

### Feature

- 搜尋功能，圖戰就是要嘴得快

---

### Built With

- Javascript
- Chrome extensions

--- 

## Demo
1. - 對圖片按右鍵 > 好圖，我要啦！！  
   - Right Click Images > 好圖，我要啦！！

2. - `存位置我的書籤/ImgurImageFight/.`
   - `yourBookmarks/ImgurImageFight/.`

--- 

### Installation

- [Chrome Extensions](chrome://extensions/)
