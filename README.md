## About The Project

- 將喜歡的Imgur圖片儲存到你的書籤列中
- Save Imgur image URL to your bookmarks.
- 將在十一月上架哦！
---

### Built With

- Chrome Extenstion built on web technologies such as HTML, JavaScript, and CSS.

--- 

## Demo
1. - 對圖片按右鍵 > 好圖，我要啦！！  
   - Right Click Images > 好圖，我要啦！！

2. - `存位置我的書籤/Imgurs/.`
   - `yourBookmarks/Imgurs/.`
